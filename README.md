## Description

This repository explains Dependency Injection in details. It covers Dependency Injection, Inversion of Control, Decorators and how the work specifically within the NestJS Architecture.

To start up the application run the command below:

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
